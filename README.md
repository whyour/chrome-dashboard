Chrome 安装
----
1. 下载项目的 zip 文件，在右上方有个 `Download ZIP`, 解压到本地

2. chrome 右上角的设置按钮下找到更多工具，打开`扩展程序`

3. 先选中`开发者模式`，拖拽解压文件中的 `chrome-dashboard.crx` 完成安装！
